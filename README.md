# webshop
There is a pet project which uses DBMS - MySQL with access to the database from my application using JDBC and the DAO template
There are 3 Tables in Database `webshop` -> `clients`, `products`, `order`
An application has methods to get, create, update and delete all kinds of data from Tables
All methods have a description, and their results are translate in console
Also there are 3 SQL scripts, ddl & dml or full script file `webshopfullsript`
